## Main API for HIMALKOM IPB landing page

## API Reference

### Komnews

#### Get all Komnews

```http
  GET /komnews
```

#### Get komnews by slug

```http
  GET /komnews/{slug}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `slug`      | `string` | **Required**. Slug of komnews to fetch |
