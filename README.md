## Main API for HIMALKOM IPB landing page

## Installation
```bash
composer install
```
<br>

```bash
cp .env.example .env
```
<br>

```bash
php artisan key:generate
```
<br>

```bash
php artisan migrate
```
<br>

```bash
php artisan db:seed
```
<br>

```bash
php artisan serve
```
<br>

```plaintext
http://localhost:8000/admin
```
